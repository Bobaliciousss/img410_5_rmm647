# Usage
- Call 'make' in the file directory to create a raytrace.exe
- Call raytrace.exe with the width, height, (inputFileName).scene, (outputFileName).pmm in that order
- E.g.     .\raytrace.exe 500 500 redball.scene output.ppm
- The output file will be created for you if one does not exist.
